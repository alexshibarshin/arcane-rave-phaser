# Arcane Rave Context

Этот документ фиксирует канонический двуязычный глоссарий проекта. Основной язык описания игры — русский, но для будущих задач агентам у каждого термина есть закрепленная английская форма.

| Русский | English | Значение |
| --- | --- | --- |
| стейдж | stage | Одна законченная игровая сессия внутри конкретного боевого сценария. |
| фаза сборки | build phase | Фаза между волнами, в которой игрок покупает, мержит и перестраивает билд. |
| боевая фаза | combat phase | Фаза реального времени, в которой пластинка крутится, идут враги, а игрок может использовать управление временем. |
| волна | wave | Конкретный состав врагов, который приходит в одной боевой фазе. |
| суб-волна | sub-wave | Внутренняя пачка врагов внутри одной волны, создающая ритм наката и отката давления. |
| боссовая волна | boss wave | Финальная волна стейджа, в которой появляется босс как главная проверка билда. |
| база | base | Системный термин для объекта защиты игрока. В художественной метафоре это `DJ booth / sound system`. |
| враг | enemy | Любая вражеская сущность, которая движется к `base` и создает боевое давление. |
| пластинка | record | Основная боевая поверхность, на которой расположены слоты и пешки. |
| иголка | needle | Фиксированная точка активации, в которую доезжают слоты на вращающейся пластинке. |
| слот | slot | Сектор на пластинке, в который можно поместить пешку. |
| особый эффект слота | slot modifier | Рандомный модификатор слота, который влияет только на часть пешек или паттернов их работы. |
| пешка | pawn | Боевая сущность, которую игрок покупает, ставит на пластинку и мержит. |
| генератор | generator | Тип пешки, который создает ноты своего цвета. |
| финишер | finisher | Тип пешки, который поглощает ноты своего цвета, усиливается от их количества и выпускает выходную ноту по своему правилу. |
| нота | note | Переносимый боевой ресурс цвета, который создают `generator` и используют `finisher`. |
| пакет нот | note packet | Текущий переносимый набор нот одного цвета, который переходит от активации к активации. |
| сегмент | segment | Последовательность соседних активаций на пластинке, работающая как локальная синергетическая связка. |
| стихия | element | Цветовая и стихийная ось урона, нот, пешек и врагов. |
| колода | deck | Предматчевый набор доступных пешек, из которого строится ассортимент `shop`. |
| магазин | shop | Источник случайных пешек, доступных для покупки в `build phase`. |
| монеты | coins | Внутристейджевая валюта для покупки пешек, реролла и перестановок. |
| доход за сбережения | investment income | Дополнительный доход между волнами за часть не потраченных `coins`. |
| Хроно | Chrono | Ресурс управления временем для `Rewind` и `FastForward`. |
| перемотка назад | Rewind | Боевое действие, которое быстро двигает пластинку назад без повторной активации пешек на обратном пути. |
| ускорение вперед | FastForward | Боевое действие, которое ускоряет вращение пластинки вперед и повышает частоту активаций. |
| мерж | merge | Объединение двух одинаковых пешек одного `tier` в усиленный результат по текущему правилу прототипа. |
| тир | tier | Уровень пешки внутри одного стейджа. |
| реролл | reroll | Обновление предложения `shop` за монеты. |
| перестановка | repositioning | Общий русский зонтичный термин для перемещения пешек между слотами на пластинке. |
| обмен местами | swap | Перестановка, при которой две пешки меняются слотами. |
| перенос в пустой слот | reposition | Перестановка, при которой пешка переезжает в другой свободный слот. |
| пустой такт | empty beat | Срабатывание пустого `slot`, которое не дает эффекта, но все равно тратит время активации. |
| превью стейджа | stage preview | Обзор состава врагов и типа давления перед входом в `stage`. |
| превью следующей волны | next wave preview | Краткое описание состава ближайшей волны перед ее стартом. |
| слабость | weakness | Выгодный стихийный матчап, дающий бонусный урон по правилу цветового треугольника. |
| контроль времени | time control | Общий термин для боевых действий игрока через `Rewind`, `FastForward` и ожидание. |
| ожидание | wait | Состояние, в котором игрок не использует активное ускорение или перемотку и восстанавливает `Chrono` по базовым правилам. |
| диджейский пульт | DJ booth | Художественная трактовка `base` внутри визуальной метафоры игры. |
| звуковая система | sound system | Альтернативная художественная трактовка `base` внутри той же метафоры. |
| капибара-диджей | DJ capybara | Главный персонаж-символ игры, продающий стиль, вайб и реактивный визуальный фидбек. |
